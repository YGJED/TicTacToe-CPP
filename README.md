# Tic Tac Toe in C++

A simple two-player Tic Tac Toe game built in C++.

## How to Compile and Run

### Prerequisites
- A C++ compiler (e.g., `clang++` or `g++`).
- Optional: `make` (to use the provided Makefile).

### Compile and Run the Game
#### Using `make`:
```bash
make
./main

#### Without Using `make`:
clang++ main.cpp -o main
./main
